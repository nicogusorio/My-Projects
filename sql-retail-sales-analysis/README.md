# SQL Retail Sales Analysis

This project analyzes retail sales data using SQL to identify key business insights.

## Objectives
- Identify top customers
- Analyze sales by product category
- Evaluate regional sales performance
- Assess category profitability

## SQL Techniques Used
- JOIN
- GROUP BY
- Aggregation (SUM)
- ORDER BY
- LIMIT
- Table creation

## Dataset
Sample Superstore dataset.

## Tools
- SQL
- SQLite
- DB Browser for SQLite
